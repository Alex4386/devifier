## □ 제작 계기 
기본 제공 HamoniKR-ME 환경의 노후화로, 최신 개방형 SW 개발 환경 구축이 어려운 문제에 대한 해결방안을 강구함.

## □ 구동 방법
1. 기본 탑재된 Visual Studio Code 실행
2. <key>Ctrl</key>+<key>Shift</key>+<key>\`</key> 으로 `Open Terminal` 명령 수행
3. `curl -L https://s4a.it/devifier | bash` 실행

## □ 탑재된 개방형 SW
* Mozilla Firefox
* Visual Studio Code
